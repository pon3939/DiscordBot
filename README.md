# DiscordBot
DiscordのBot  
だれか面白いBOT作って
___
## チャット
### google画像検索
@{BOT} gogle search {検索キーワード}でgoogle画像検索
### 独りぼっち通知メッセージ編集
`@{BOT} lonely list`で独りぼっち通知メッセージの一覧表示  
`@{BOT} lonely add {追加したいテンプレート}`で独りぼっち通知メッセージ追加  
`@{BOT} lonely delete {削除したいテンプレートのインデックス}`で独りぼっち通知メッセージ削除
___
## 通知
### 独りぼっち通知
ボイスチャンネルに誰かが独りぼっちだと、チャットで通知する
